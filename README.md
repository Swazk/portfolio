# My Portfolio
_**Personal Portfolio Website**_
A modern, interactive personal portfolio showcasing my projects, skills, and experience, built with smooth transitions and engaging UI effects.

_**Live Demo**_

*  GitHub Repository: This repo
 
_**Features**_
* Smooth page transitions and interactive UI
* Responsive design (Desktop, Tablet & Mobile)
* Clean and minimal layout
* Project showcase with live links
* Contact & social media integration

_**Tech Stack**_
* Frontend: HTML, CSS, JavaScript
* Animations: CSS Transitions, JavaScript effects
* Version Control: Git & GitHub
*  Deployment: GitHub Pages / Netlify / Vercel
